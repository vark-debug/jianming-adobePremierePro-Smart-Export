## Version 1.2.0 (2026-02-28)

### 🎉 归档与备份能力上线（迁移阶段收官版本）

### ✨ 新特性
- **定稿版自动归档**：支持设置归档根目录与目录模板（`YYYY`/`MM`/`DD`/`项目名称`）
- **导出前自动备份**：支持序列克隆备份与工程文件二进制备份
- **设置项扩展**：新增归档、备份相关持久化设置
- **界面与文案更新**：设置页与多语言文案新增归档/备份配置入口

### 🔧 技术改进
- 新增 `archiveManager.ts`，实现模板解析、路径预览、分层目录创建与文件归档
- 新增 `preExportBackup.ts`，实现导出前序列和工程文件备份流程
- 在导出主流程中串联“备份 → 导出 → 归档”能力，失败不阻断核心导出

## Version 1.0.0 (2026-01-21)

### 🎉 Initial Release - Migration to Bolt UXP

完全重构的现代化版本，从原始 CommonJS 插件迁移到 Bolt UXP 框架。

### ✨ 新特性
- **现代化技术栈**：Vue 3 + TypeScript + Vite
- **模块化架构**：清晰的模块划分与类型安全
- **热重载开发**：支持开发期快速迭代

### ✅ 核心功能
- 智能分辨率检测（4K/1080p 自动选择码率）
- 自动版本号管理（数字版/中文版）
- 调色状态标记（已调色/未调色）
- 自定义项目名称
- 多格式支持（H.264 / ProRes 422 / ProRes 444）
- 智能文件命名（自动清理日期标记）
- 导出文件夹自动创建与管理

### 🔧 技术改进
- 完整 TypeScript 类型定义
- Promise 异步流程
- 错误处理与日志追踪增强
- 跨平台路径处理优化

### 📋 兼容性
- Adobe Premiere Pro 25.6.3+
- 开发调试需使用 UXP Developer Tool（UDT）
